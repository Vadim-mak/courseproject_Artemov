# Этап 7: Пользовательский интерфейс

## Экраны мобильного приложения

Мобильный клиент реализован на **React Native + Expo** и содержит 9 экранов
(требование методички: 5+). Интерфейс построен по принципам **Material Design**,
использует единую дизайн-систему из `src/theme/theme.ts`.

### Палитра дизайна

| Токен | Цвет | Назначение |
|---|---|---|
| `primary` | `#0E3B43` (океанский тил) | Шапки, кнопки, иконки |
| `accent` | `#FF6B4A` (закатный коралл) | CTA, активные состояния |
| `sand` | `#F4E9D8` | Карточки, фон чипов |
| `background` | `#F7F4EF` | Фон страниц |

### Структура навигации

```
RootNavigator
├── AuthNavigator (не аутентифицирован)
│   ├── LoginScreen          — вход
│   └── RegisterScreen       — регистрация
│
└── AppNavigator (аутентифицирован)
    ├── Tours (Tab)
    │   ├── HomeScreen        — главная: направления + ТОП туров
    │   ├── TourListScreen    — каталог с поиском
    │   ├── TourDetailScreen  — детали тура, отзывы
    │   └── BookingScreen     — оформление бронирования
    ├── Bookings (Tab)
    │   └── MyBookingsScreen  — история поездок
    ├── Profile (Tab)
    │   └── ProfileScreen     — профиль, редактирование, выход
    └── Admin (Tab, только ROLE_ADMIN)
        └── AdminScreen       — панель администратора
```

### Описание экранов

#### LoginScreen
- Фирменная шапка с логотипом и слоганом
- Поля email / пароль с валидацией
- Ссылка на регистрацию

#### RegisterScreen
- Форма: имя, email, телефон, пароль, подтверждение
- Клиентская валидация (длина, формат email, совпадение паролей)

#### HomeScreen
- Приветствие с именем пользователя
- Горизонтальный скролл направлений (Country chips)
- Список популярных туров (TourCard)

#### TourListScreen
- Строка поиска с иконкой и кнопкой сброса
- FlatList с ленивой подгрузкой (onEndReached)
- EmptyState при отсутствии результатов

#### TourDetailScreen
- Hero-изображение на всю ширину
- Блок статистики (4 плитки: начало, конец, дней, мест)
- Описание тура
- Блок отзывов со StarRating
- Закреплённый footer с ценой и кнопкой бронирования

#### BookingScreen
- Карточка тура (краткая информация)
- Счётчик туристов (+/−)
- Блок итоговой стоимости с разбивкой
- Кнопка подтверждения

#### MyBookingsScreen
- FlatList бронирований
- Цветовые бейджи статусов (жёлтый / зелёный / красный)
- Кнопка отмены для активных бронирований

#### ProfileScreen
- Аватар с первой буквой имени
- Бейдж «Администратор» для ROLE_ADMIN
- Карточка с информацией / режим редактирования
- Кнопка выхода

#### AdminScreen
- Статистика (3 счётчика)
- Переключатель вкладок: Бронирования / Пользователи
- Карточки бронирований с кнопками «Подтвердить» / «Отменить»
- Список пользователей с ролями
